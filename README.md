## Installation

```
$ git clone git@github.com:BoisselNicolas/PenduCPP.git
```

## Compilation


```
$ g++ main.cpp -o main
```
Le paramètre "-o" définit le nom qu'aura notre éxécutable.

## Usage 

Pour éxécuter le programme

```
$ ./main
```
